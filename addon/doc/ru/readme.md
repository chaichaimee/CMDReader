## Хватит гадать. Пора читать.
Работа с командной строкой Windows (CMD) или PowerShell может быть настоящим хаосом. **CMDReader** превращает этот хаос в ясность.

### Интеллектуальные функции
* **Мониторинг в реальном времени:** CMDReader автоматически озвучивает новую информацию по мере появления.
* **Умная фильтрация шума:** Наш алгоритм удаляет лишние пробелы и ненужные декоративные элементы.
* **Оповещения об ошибках:** Подает особый звуковой сигнал, как только что-то идет не так.
* **Сигнал о завершении:** Уведомит вас фразой «Command Finished» в секунду завершения команды.
* **Чистая теневая история:** Построчная навигация по истории с помощью специального буфера.

---

## Навигация
1.  **Мониторинг:** Просто перейдите в любое окно CMD или PowerShell.
2.  **Прокрутка истории:** Используйте `Ctrl+Alt+Стрелка вверх` или `вниз` для перемещения без сдвига курсора.
3.  **Анализ логов:** Нажмите `Ctrl+Alt+L`, чтобы открыть идеально отформатированный текстовый файл.

### Быстрая справка
| Клавиши | Действие | Результат |
| :--- | :--- | :--- |
| `Ctrl+Alt+Стрелка вверх` | Предыдущая строка | Читает предыдущую строку |
| `Ctrl+Alt+Стрелка вниз` | Следующая строка | Читает следующую строку |
| `Ctrl+Alt+L` | Открыть лог | Открывает очищенный лог в редакторе |
| (Автоматически) | Ошибки | Особый сигнал (440 Гц) при ошибках |
| (Автоматически) | Завершено | Сигнал «Command Finished» (880 Гц) |
