# CMDReader
### *Barrierefreiheit auf der Befehlszeile, neu definiert.*

**Autor:** chai chaimee  
**URL:** [github.com/chaichaimee/CMDReader](https://github.com/chaichaimee/CMDReader)

---

## Schluss mit dem Raten. Fangen Sie an zu lesen.
Das Arbeiten mit der Windows-Eingabeaufforderung (CMD) oder PowerShell kann für Screenreader-Benutzer chaotisch sein. **CMDReader** verwandelt dieses Chaos in Klarheit. Kein mühsames Suchen mehr nach dem Start der Ausgabe oder die Frage, ob ein Prozess abgeschlossen ist.

### Intelligente Funktionen
* **Echtzeit-Ausgabeüberwachung:** Lehnen Sie sich zurück. CMDReader spricht neue Informationen automatisch aus, sobald sie erscheinen.
* **Intelligente Rauschfilterung:** Verabschieden Sie sich von überladen Ausgaben. Unsere Logik entfernt Leerzeichen am Zeilenende.
* **Sofortige Fehlerwarnungen:** CMDReader achtet auf gängige Fehler-Keywords und warnt Sie mit einem markanten Ton.
* **Signal bei Prozessabschluss:** Bei langwierigen Aufgaben benachrichtigt Sie CMDReader mit der Meldung "Command Finished".
* **Sauberer Schattenverlauf:** Navigieren Sie durch jede Zeile Ihres Sitzungsverlaufs mit einem dedizierten "Shadow Buffer".

---

## Navigation
1.  **Überwachung:** Fokussieren Sie einfach ein CMD- oder PowerShell-Fenster. CMDReader beginnt automatisch mit der Arbeit.
2.  **Verlauf durchblättern:** Verwenden Sie `Strg+Alt+Oben` oder `Unten`, um sich zeilenweise durch die Sitzungsausgabe zu bewegen.
3.  **Protokolle analysieren:** Drücken Sie `Strg+Alt+L`, um eine perfekt formatierte Textdatei Ihrer aktuellen Sitzung zu öffnen.

### Kurzübersicht
| Geste | Aktion | Ergebnis |
| :--- | :--- | :--- |
| `Strg+Alt+Pfeil oben` | Vorherige Zeile | Liest die vorherige Zeile im Verlauf |
| `Strg+Alt+Pfeil unten` | Nächste Zeile | Liest die nächste Zeile im Verlauf |
| `Strg+Alt+L` | Sitzungsprotokoll | Öffnet das bereinigte Protokoll |
| (Automatisch) | Fehlererkennung | Markanter Signalton (440Hz) bei Fehlern |
| (Automatisch) | Auftrag erledigt | "Command Finished" Signalton (880Hz) |
