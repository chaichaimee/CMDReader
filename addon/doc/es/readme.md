# CMDReader

**Autor:** chai chaimee  
**URL:** [github.com/chaichaimee/CMDReader](https://github.com/chaichaimee/CMDReader)

## Deje de adivinar. Empiece a leer.
Trabajar con el Símbolo del sistema de Windows (CMD) o PowerShell puede ser una experiencia caótica para los usuarios de lectores de pantalla. **CMDReader** transforma ese caos en claridad.

### Funciones Inteligentes
* **Monitorización en tiempo real:** CMDReader habla automáticamente la nueva información a medida que aparece.
* **Filtrado inteligente de ruido:** Nuestra lógica elimina los espacios al final de las líneas y las líneas decorativas inútiles.
* **Alertas de error instantáneas:** CMDReader escucha palabras clave de error comunes y le alerta con un tono distintivo.
* **Señal de finalización:** CMDReader le notificará con un mensaje de "Command Finished" cuando la tarea termine.
* **Historial de sombras limpio:** Navegue por cada línea de su historial utilizando un "Búfer de Sombra" dedicado.

---

## Cómo navegar
1.  **Monitorización:** Simplemente enfoque cualquier ventana de CMD o PowerShell.
2.  **Desplazamiento:** Use `Ctrl+Alt+Flecha Arriba` o `Abajo` para moverse línea por línea.
3.  **Registros:** Pulse `Ctrl+Alt+L` para abrir un archivo de registro perfectamente formateado.

### Resumen de referencia rápida
| Gesto | Acción | Resultado |
| :--- | :--- | :--- |
| `Ctrl+Alt+Flecha Arriba` | Línea anterior | Lee la línea anterior en el historial |
| `Ctrl+Alt+Flecha Abajo` | Línea siguiente | Lee la línea siguiente en el historial |
| `Ctrl+Alt+L` | Ver registro | Abre el registro limpio en el editor |
| (Automático) | Errores | Pitido distintivo (440Hz) en errores |
| (Automático) | Completado | Timbre "Command Finished" (880Hz) |
