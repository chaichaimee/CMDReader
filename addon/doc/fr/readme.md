# CMDReader
### *L'accessibilité en ligne de commande, redéfinie.*

**Auteur :** chai chaimee  
**URL :** [github.com/chaichaimee/CMDReader](https://github.com/chaichaimee/CMDReader)

---

## Arrêtez de deviner. Commencez à lire.
Travailler avec l'invite de commande Windows (CMD) ou PowerShell peut être une expérience chaotique. **CMDReader** transforme ce chaos en clarté.

### Fonctionnalités Intelligentes
* **Surveillance en temps réel :** CMDReader énonce automatiquement les nouvelles informations au fur et à mesure.
* **Filtrage intelligent :** Notre logique supprime les espaces en fin de ligne et les lignes inutiles.
* **Alertes d'erreur :** CMDReader écoute les mots-clés d'erreur et vous alerte avec une tonalité distincte.
* **Signal de fin :** CMDReader vous informera avec un message "Command Finished" dès que l'invite revient.
* **Historique fantôme :** Naviguez à travers chaque ligne de votre session avec un tampon dédié.

---

## Comment naviguer
1.  **Surveillance :** Donnez simplement le focus à une fenêtre CMD ou PowerShell.
2.  **Défilement :** Utilisez `Ctrl+Alt+Flèche Haut` ou `Bas` pour vous déplacer ligne par ligne.
3.  **Journaux :** Appuyez sur `Ctrl+Alt+L` pour ouvrir un journal parfaitement formaté.

### Résumé de référence
| Geste | Action | Résultat |
| :--- | :--- | :--- |
| `Ctrl+Alt+Flèche Haut` | Ligne précédente | Lit la ligne précédente |
| `Ctrl+Alt+Flèche Bas` | Ligne suivante | Lit la ligne suivante |
| `Ctrl+Alt+L` | Voir le journal | Ouvre le journal dans l'éditeur |
| (Automatique) | Erreurs | Bip distinct (440Hz) |
| (Automatique) | Terminée | Carillon "Command Finished" (880Hz) |
